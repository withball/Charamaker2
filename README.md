# Charamaker2

これはC#で2Dゲームを作成する際に便利なクラスとか命令とかのセットです。

[できること]
・bmp画像の表示、回転、透明化
・文字の表示、回転、透明化、フェード。フォントは固定。
・bmp画像の集合体Characterとそれを動かすmotionの作成
・Shapeを用いたあたり判定の作成
